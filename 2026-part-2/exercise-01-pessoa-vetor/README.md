# Exercise 01 - pessoa vetor

## Description

- Crie uma classe Pessoa com id, nome, salario encapsulados e sobreescreva o método ToString para mostrar o id e o nome da pessoa. Sobreescreva o método Equals e HashCode na classe Pessoa.

- Crie uma classe Principal com um vetor de pessoas e adicione 10 pessoas com id, nome e salário no vetor.

- Crie uma método na classe Principal para informar qual é a pessoa com o maior salario.

- Crie uma método na classe Principal para informar qual é a pessoa com o menor salario.

- Crie um método na classe Principal que retorne um objeto pessoa pelo id.

- Crie um menu para orquestrar todas as opções (inserir, mostrar maior, mostrar menor, pesquisar pessoa)
